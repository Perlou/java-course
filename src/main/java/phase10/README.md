## 🎯 学习目标

## 📚 核心概念

### 服务治理

- 服务注册与发现（Nacos/Eureka）
- 负载均衡（LoadBalancer）
- 服务调用（OpenFeign）

### 高可用

- 熔断降级（Sentinel）
- 限流
- 服务网关（Gateway）
