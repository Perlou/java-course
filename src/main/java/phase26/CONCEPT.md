## 📊 向量嵌入

### 什么是向量嵌入

```
文本 → 向量表示

"Java 是编程语言" → [0.12, -0.34, 0.56, ..., 0.89]  (1536维)
"Python 是脚本语言" → [0.11, -0.32, 0.55, ..., 0.87]

相似文本 → 向量距离近
不同文本 → 向量距离远

计算相似度：余弦相似度 = A·B / (|A|×|B|)
```

### 向量数据库对比

| 数据库       | 特点            | 适用场景       |
| ------------ | --------------- | -------------- |
| **Milvus**   | 高性能，分布式  | 大规模生产环境 |
| **pgvector** | PostgreSQL 扩展 | 已有 PG 环境   |
| **Chroma**   | 轻量，易上手    | 开发测试       |
| **Pinecone** | 全托管          | 快速上线       |

## ✂️ 文档切分策略

```
切分策略对比
═════════════════════════════════════════════════════════

固定大小切分:
┌────────────────────────────────────────┐
│ Chunk 1 (500 tokens) │ Chunk 2 (500)  │
└────────────────────────────────────────┘
简单但可能切断语义

语义切分:
┌────────────────────────────────────────┐
│ 段落1 │ 段落2 │ 段落3 │ 段落4         │
└────────────────────────────────────────┘
保持语义完整

重叠切分:
┌──────────────────────────────┐
│ Chunk 1                      │
│         └──────────┐         │
│           Chunk 2  │         │
└──────────────────────────────┘
overlap = 50 tokens，保证上下文连续
```

## 🔍 检索策略

| 策略           | 说明               | 优劣                       |
| -------------- | ------------------ | -------------------------- |
| **向量检索**   | 语义相似度搜索     | 理解语义，但可能遗漏关键词 |
| **关键词检索** | BM25 等传统搜索    | 精确匹配，但不理解语义     |
| **混合检索**   | 向量 + 关键词结合  | 综合优势，推荐使用         |
| **重排序**     | 对检索结果二次排序 | 提高相关性                 |

## 🏢 企业应用最佳实践

### 成本控制

```
成本优化策略
═════════════════════════════════════════════════════════

1. 模型分级
   简单问题 → GPT-3.5 (便宜)
   复杂问题 → GPT-4 (昂贵)

2. 缓存策略
   相似问题 → 返回缓存结果

3. Prompt 优化
   减少不必要的 token 消耗

4. 批量处理
   非实时场景使用 Batch API
```

### 安全合规

| 风险            | 措施                   |
| --------------- | ---------------------- |
| **数据泄露**    | 敏感数据脱敏后再送模型 |
| **Prompt 注入** | 输入过滤、角色限制     |
| **幻觉问题**    | RAG 增强、事实核查     |
| **合规审计**    | 日志记录、监控告警     |

## 🔧 Spring AI 集成

```java
// Spring AI 配置
@Configuration
public class AiConfig {
    @Bean
    public ChatClient chatClient(OpenAiChatModel model) {
        return ChatClient.create(model);
    }
}

// 使用示例
@RestController
public class ChatController {
    @Autowired
    private ChatClient chatClient;

    @PostMapping("/chat")
    public String chat(@RequestBody String message) {
        return chatClient.prompt()
            .user(message)
            .call()
            .content();
    }
}
```
