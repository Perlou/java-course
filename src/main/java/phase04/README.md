## 📚 核心概念

### IO 流分类

- 按方向：输入流、输出流
- 按数据：字节流、字符流
- 按功能：节点流、处理流

### NIO 核心组件

- Buffer：数据缓冲区
- Channel：数据通道
- Selector：多路复用器
