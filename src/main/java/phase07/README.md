# Phase 7: 设计模式

> **目标**：掌握常用设计模式与 SOLID 原则  
> **预计时长**：2 周  
> **前置条件**：Phase 6 完成

---

## 🎯 学习目标

完成本阶段后，你将能够：

1. 理解 23 种设计模式
2. 掌握创建型、结构型、行为型模式
3. 理解 SOLID 设计原则
4. 能够在实际项目中应用设计模式

---

## 📚 核心概念

### 创建型模式

- 单例、工厂、抽象工厂、建造者、原型

### 结构型模式

- 代理、装饰器、适配器、外观、桥接、组合、享元

### 行为型模式

- 策略、观察者、模板方法、责任链、状态、命令、迭代器、中介者、备忘录、访问者
