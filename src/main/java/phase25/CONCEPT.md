## 📝 Prompt Engineering

### 核心原则

| 原则         | 说明              | 示例                                |
| ------------ | ----------------- | ----------------------------------- |
| **清晰具体** | 明确说明任务要求  | "用中文总结以下文章，不超过 100 字" |
| **角色设定** | 赋予 AI 特定身份  | "你是一个资深 Java 架构师"          |
| **示例引导** | Few-shot Learning | 提供 2-3 个示例                     |
| **格式约束** | 指定输出格式      | "以 JSON 格式返回"                  |
| **分步思考** | Chain of Thought  | "让我们一步步分析这个问题"          |

### Prompt 模板

```
角色: 你是一个{role}

任务: {task_description}

输入: {user_input}

要求:
1. {requirement_1}
2. {requirement_2}

输出格式: {format}
```

### 代码示例

```java
// 基本对话
ChatLanguageModel model = OpenAiChatModel.builder()
    .apiKey(apiKey)
    .modelName("gpt-4")
    .build();

String response = model.generate("Java 和 Python 的区别？");

// AI Service（推荐方式）
interface Assistant {
    @SystemMessage("你是一个友好的助手")
    String chat(String message);
}

Assistant assistant = AiServices.create(Assistant.class, model);
String answer = assistant.chat("你好");
```

## 💰 Token 与成本

| 模型        | 输入价格  | 输出价格  | 上下文长度 |
| ----------- | --------- | --------- | ---------- |
| GPT-4 Turbo | $10/1M    | $30/1M    | 128K       |
| GPT-3.5     | $0.5/1M   | $1.5/1M   | 16K        |
| Claude 3    | $3/1M     | $15/1M    | 200K       |
| 通义千问    | ¥0.008/1K | ¥0.008/1K | 128K       |

| 模型         | 优势                | 适用场景           |
