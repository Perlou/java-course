## 1. 产品概述

### 1.1 产品定位

TurboX Exchange 是一个面向数字资产的交易平台，提供现货交易功能，支持多种交易对的撮合、资金管理和行情展示。

### 1.2 目标用户

| 用户类型   | 描述                   |
| ---------- | ---------------------- |
| 普通交易者 | 进行买卖操作的终端用户 |
| 做市商     | 提供流动性的专业交易者 |
| 平台管理员 | 管理平台运营的后台人员 |

### 1.3 核心价值

- **高效**：毫秒级撮合，实时行情
- **安全**：资金安全，风控完善
- **稳定**：高可用架构，99.9% SLA

## 3. 非功能需求

### 3.1 性能需求

| 指标         | 要求               |
| ------------ | ------------------ |
| API 响应时间 | P99 < 100ms        |
| 撮合延迟     | < 1ms              |
| 行情推送延迟 | < 10ms             |
| 系统吞吐量   | > 10,000 QPS (API) |

### 3.2 可用性需求

| 指标         | 要求           |
| ------------ | -------------- |
| 系统可用性   | 99.9%          |
| 计划外停机   | < 8.76 小时/年 |
| 故障恢复时间 | RTO < 5 分钟   |
| 数据恢复点   | RPO < 1 分钟   |

### 3.3 安全需求

| 类别     | 要求             |
| -------- | ---------------- |
| 数据传输 | 全站 HTTPS       |
| 密码存储 | BCrypt 加密      |
| 敏感操作 | 二次验证         |
| API 安全 | 签名验证、防重放 |
| 审计日志 | 关键操作留痕     |

### 3.4 可扩展性

| 类别     | 要求                   |
| -------- | ---------------------- |
| 水平扩展 | 无状态服务支持横向扩展 |
| 撮合引擎 | 支持多交易对并行       |
| 存储扩展 | 支持分库分表           |

## 5. 术语表

| 术语   | 说明                            |
| ------ | ------------------------------- |
| 交易对 | 两种资产的交易组合，如 BTC/USDT |
| 限价单 | 指定价格的订单                  |
| 市价单 | 按市场最优价成交的订单          |
| Maker  | 挂单方，提供流动性              |
| Taker  | 吃单方，消耗流动性              |
| 盘口   | 买卖订单的价格和数量列表        |
| 深度   | 各价位的订单累计数量            |
| K 线   | 一定时间周期内的开高低收价格图  |
| 2FA    | 双因素认证                      |
| KYC    | Know Your Customer，实名认证    |
